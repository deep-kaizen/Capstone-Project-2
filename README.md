# Capstone Project 2 - S3 Static Website using Terraform
This project provisions a **publicly accessible static website** on Amazon S3 using Terraform.

## What it does
- Creates an S3 bucket
- Enables static website hosting
- Allows public read access
- Uploads `index.html`

## Prerequisites
- AWS account
- Terraform installed
- AWS credentials configured

## Usage
```bash
terraform init
terraform apply
```
